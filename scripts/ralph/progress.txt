## Codebase Patterns (Updated 2026-03-02)
- Prisma schema engine binary cannot be downloaded in this environment (network restriction to binaries.prisma.sh). Generated client files must be updated manually after schema changes: update `enums.ts`, `models/User.ts`, `commonInputTypes.ts`, `internal/prismaNamespace.ts`, and `internal/class.ts` (inlineSchema).
- When adding a new enum field to a model, update ALL type definitions in the generated model file: aggregate types (Min/Max/Count), where filters, create inputs, update inputs, orderBy, scalars in $Payload, and FieldRef. Also add corresponding filter types in `commonInputTypes.ts` and field ref types in `prismaNamespace.ts`.
